{# Shared vessel picker (Movement / Press / Rack / Bottling / Blend). Any move into a tank another lot occupies is now a blend, not a plain transfer — combining two wines without writing the LotLineage edge used to leave no compliance record and only show the last-moved lot on the tank map. So a plain transfer/press/rack/bottling destination is never allowed to be occupied: occupied options stay disabled here, full stop. The one exception is the Blend form's own "to vessel" picker (`blend=True`), which legitimately targets an already-occupied tank by design — those options are always selectable there, no checkbox needed. Expects: options (from web.vessels.vessel_options), field (select name), picker_id, and optionally blend=True for the Blend card's own picker. #}
